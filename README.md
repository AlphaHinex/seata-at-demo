Seata AT Mode Demo
==================

How to use?
-----------

1. Build service jars

    ```
    $ ./gradlew bootJar
    ```

1. Docker compose

    ```
    $ docker-compose up -d
    ``` 

1. Wait for all services boot up
   
1. Access Nacos with `nacos`/`nacos`

    http://localhost:8848/nacos
